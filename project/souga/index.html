سوجا Souga — تطبيق تجارة إلكترونية عربي

سوق إلكتروني عربي بتقنيات متطورة — تحميل مجاني

Android منشور ✓ FlutterFirebaseDartImageKitGemini AI

منصة تجارة إلكترونية محلية متكاملة ببنية ثلاثية الأدوار: مشتري، بائع، مدير — مبنية بـ Flutter وFirebase.

  • سوجا ليست مجرد تطبيق تسوق، بل منصة تجارية متكاملة أُعدت لتكون بديلاً محلياً حقيقياً لمنصات التجارة الكبرى.
  • دردشة مباشرة داخل التطبيق بين المشتري والبائع دون الحاجة لتطبيقات خارجية.
  • ذكاء اصطناعي Gemini لكتابة إعلانات تسويقية جذابة بضغطة واحدة، ورفع صور المنتجات عبر ImageKit بأعلى جودة وأسرع تحميل.
  • نظام طلبات متكامل مع تتبع الحالة لحظة بلحظة، وإشعارات فورية بكل تحديث، ودعم 42 عملة عالمية مع تحويل آني.
  • لوحة تحكم للمدير بصلاحيات شاملة ومرنة، وتسجيل دخول آمن عبر Firebase Auth مع تسجيل الدخول بحساب Google.
