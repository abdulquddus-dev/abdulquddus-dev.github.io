ShaibAI Desktop — ذكاء اصطناعي محلي على Windows بدون إنترنت

ذكاء اصطناعي محلي كامل بدون إنترنت — Windows

Desktop منشور ✓ C#.NET 4.8WinFormsLM StudioQwen2.5

برنامج Windows يتصل بـ LM Studio لتشغيل نماذج الذكاء الاصطناعي محلياً بالكامل بدون إنترنت، مبني بـ C# و.NET WinForms.

  • ShaibAI جاء من قناعة شخصية: الخصوصية حق وليست رفاهية.
  • يعمل بالكامل بدون إنترنت عبر LM Studio، ويدعم نموذج Qwen2.5-7B-Instruct بشكل مثالي على المعالج دون الحاجة لكارت شاشة.
  • استجابات تتدفق في الوقت الفعلي (Streaming)، وشريط محادثات جانبي لتنظيم المحادثات السابقة.
  • واجهة ثنائية اللغة تدعم العربية والإنجليزية مع معالجة نصوص عربية محسّنة.
  • اختيار النموذج وتحميله بضغطة واحدة من داخل البرنامج.
