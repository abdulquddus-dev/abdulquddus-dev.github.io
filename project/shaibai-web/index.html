نسخة ASP.NET WebForms من ShaibAI تعمل من المتصفح وتتصل بـ LM Studio محلياً بدون إرسال أي بيانات للإنترنت.

  • ShaibAI Web هي النسخة التي تعمل من المتصفح مباشرة — بدون تثبيت، وبدون إعداد معقد.
  • تعمل على Chrome وFirefox وEdge بدون مشكلة، وتتصل بـ LM Studio عبر REST API محلي على نفس الجهاز.
  • استجابات تتدفق في الوقت الفعلي، وواجهة RTL عربية بالكامل، وتاريخ محادثات منظم.
  • نسخ ومشاركة الردود في خطوة واحدة، وتصميم متجاوب يعمل على سطح المكتب والموبايل.
